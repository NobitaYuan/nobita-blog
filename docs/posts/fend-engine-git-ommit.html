需要的工具 #

commitlint #

Github | 官方文档

检查提交消息是否符合常规提交格式

cz-git #

Github | 官方文档

工程性更强，轻量级，高度自定义， 输出标准格式的 Commitizen 适配器和 CLI

可与commitlint配合

快速一套配置 #

  1. 安装 commitizen
pnpm i -d commitizen
  1. 安装 cz-git
pnpm install -D cz-git
  1. 配置package.json
...
"scripts": {
    ...
    "cz": "cz"
}
"config":{
    "commitizen": {
        "path": "node_modules/cz-git",
    }
},
  1. 添加配置文件.commitlintrc.js
